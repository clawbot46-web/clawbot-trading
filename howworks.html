How It Works

A complete breakdown of the Clawbot system — from the data it reads, to the decisions it makes, to the trades it fires every hour.

Project Overview

The big picture — what Clawbot is, why it was built, and what questions it's trying to answer.

🤖 What Is Clawbot?
  • An automated stock & crypto trading system built in Python
  • 3 bots with completely different personalities, all trading the same markets
  • Each bot starts with $1,000,000 in Alpaca paper money (no real cash at risk)
  • Bots run 24/7, making decisions and placing trades while you sleep
🎯 Project Objectives
  • Can AI-driven strategies actually beat the S&P 500?
  • Does more risk = more reward? (spoiler: maybe not)
  • Can a completely random bot beat a carefully designed algorithm?
  • Learn Python, trading APIs, and how real financial markets behave

⚡ Tech Stack

Python 3 Alpaca Trading API AI Sentiment Analysis RSI / MACD / Moving Averages Google Sheets Logging HTML + JavaScript Dashboard GitHub Pages Linux Server

What This Means

Clawbot is an experiment in automated trading. Each bot has its own rules — one chases momentum and takes big risks, one plays it safe and waits for perfect setups, and one just trades randomly with no logic at all. The goal is to run them all for 30 days and see which strategy wins. The answer might surprise you.

1

Market Data is pulled every hour

The system queries the Alpaca API for the latest prices, volume, and market conditions. A news API also pulls recent headlines, which get scored for positive or negative sentiment.

2

The AI Research Engine processes everything

Technical indicators are calculated — RSI to measure momentum, MACD to spot trend shifts, and moving averages to identify direction. Sentiment scores from the news are layered on top to give each asset an overall signal.

3

Each bot makes its own independent decision

The same signals go to all three bots, but they interpret them completely differently. Aggressive might see a bullish signal and go all in. Middle might take a small position and wait. Random flips a coin.

4

Trades fire through Alpaca's paper trading system

Each bot has its own $1M paper account — no real money involved. Alpaca simulates real market fills, so the results are as close to real trading as possible without any financial risk.

5

Results are logged and pushed to this website

Every trade gets written to a CSV log. A Python script processes those logs into a JSON file, commits it to GitHub, and this website automatically shows the updated balances, trades, and charts within the hour.

Meet the Three Bots

Same market data. Completely different decision making. That's the whole experiment.

High Risk
🔥 Aggressive

Takes big positions (up to 10% of portfolio in one trade), chases momentum signals hard, trades crypto and high-volatility stocks. Never hesitates. Maximum risk, maximum potential reward.

Typical reason: "RSI signals strong momentum — going in 10%"

Balanced
⚖️ Middle Road

Patient and disciplined. Waits for multiple indicators to line up before entering a trade. Small position sizes, strict stop-losses. Won't chase a trade — if the setup isn't perfect, it passes.

Typical reason: "RSI + MACD convergence confirmed — entering small"

Chaos Mode
🎲 Random

Completely ignores all signals. Picks stocks randomly, decides position size randomly, and gives every trade a ridiculous excuse. It exists to answer one question: can a monkey beat the algorithm?

Typical reason: "Mercury is in retrograde — seemed like a vibe"

Why This Matters

The Random Bot isn't just a joke — it's a genuine scientific control. In finance, the "random walk" theory argues that markets are so unpredictable that random selection can match expert picks. By running a truly random bot for 30 days against two carefully engineered strategies, we get a real answer to that question. If Random wins, that tells you something important about how much skill actually matters in short-term trading.
